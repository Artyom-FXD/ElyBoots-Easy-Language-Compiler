# ely Language

**ely Language** — статически типизированный компилируемый язык программирования с быстрой компиляцией, встроенными коллекциями, дженериками и удобным синтаксисом.

- Молниеносная компиляция (TCC)
- Безопасная статическая типизация
- Динамические массивы и словари
- Обобщения (дженерики)
- Обработка ошибок (`asafe`/`except`/`throw`)
- F-строки с интерполяцией
- Модульная система
- Встроенная стандартная библиотека
- Поддержка JSON через модуль DictServer

## Возможности

- **Статическая типизация** — безопасность типов на этапе компиляции. Вывод типов через `var`.  
- **Молниеносная компиляция** — благодаря Tiny C Compiler (TCC) компиляция происходит со скоростью сотни тысяч строк в секунду.  
- **Встроенные коллекции** — динамические массивы (`arr<T>`) и словари (`dict<K,V>`) с удобными методами: `push`, `pop`, `len`, доступ по индексу и ключу.  
- **Обобщения (дженерики)** — пишите обобщённые функции и структуры. Типы выводятся из аргументов. Реализована мономорфизация — каждая специализация генерирует оптимальный код.  
- **Обработка ошибок** — блоки `asafe` / `except` / `throw` — понятный механизм исключений.  
- **F-строки** — интерполяция выражений прямо в строке: `f"Привет, {name}!"`. Автоматическое преобразование в строку массивов, словарей, чисел.  
- **Модульность** — сборка проектов через `manager.json`. Модули компилируются в динамические библиотеки (`.dll`/`.so`) и могут переиспользоваться.  
- **Будущий сборщик мусора** — в следующем релизе — автоматическое управление памятью для строк, массивов, словарей и объектов.

## Синтаксис и примеры

```c
// Hello, World!
public func main() -> int {
    println("Hello, World!");
    return 0;
}
```

```c
// Обобщённая функция
func identity<T>(T x) -> T {
    return x;
}

// Массивы и словари
int[] a = [1, 2, 3];
a.push(4);
println(a);          // [1,2,3,4]

dict<str, int> d = {"x": 10, "y": 20};
d.z = 30;
println(d);          // {"x":10,"y":20,"z":30}

// Обработка ошибок
asafe {
    throw "Something went wrong";
} except (str e) {
    println(e);
}

// F-строка
str name = "World";
println(f"Hello, {name}!");   // Hello, World!
```

## Стандартная библиотека

Все встроенные функции доступны без `using` и написаны в стиле **camelCase**.

### Консоль

| Функция               | Описание                                     | Пример                              |
|-----------------------|----------------------------------------------|-------------------------------------|
| `print(str)`          | Выводит строку без перевода строки           | `print("Hello")`                    |
| `println(str)`        | Выводит строку с переводом строки            | `println("World")`                  |
| `println(int)`        | Выводит число                                | `println(42)`                       |
| `println(bool)`       | Выводит true/false                           | `println(true)`                     |
| `println(arr<T>)`     | Выводит массив в формате JSON                | `println([1,2,3])` → `[1,2,3]`      |
| `println(dict<K,V>)`  | Выводит словарь в формате JSON               | `println({"x":5})` → `{"x":5}`       |
| `input()`             | Читает строку из stdin                       | `str s = input()`                   |
| `inputPrompt(str)`    | Выводит приглашение и читает строку          | `str name = inputPrompt("Name: ")`  |

### Преобразования типов

| Функция                   | Описание                     | Пример                                      |
|---------------------------|------------------------------|---------------------------------------------|
| `strToInt(str)`           | Строка → int                 | `int x = strToInt("123")`                   |
| `strToUint(str)`          | Строка → uint                | `uint u = strToUint("42")`                  |
| `strToMore(str)`          | Строка → more (long long)    | `more m = strToMore("9223372036854775807")` |
| `strToUm(str)`            | Строка → umore               | `umore um = strToUm("18446744073709551615")`|
| `strToFlt(str)`           | Строка → flt (float)         | `flt f = strToFlt("3.14")`                  |
| `strToDouble(str)`        | Строка → double              | `double d = strToDouble("3.14159")`         |
| `intToStr(int)`           | int → строка                 | `str s = intToStr(42)`                      |
| `uintToStr(uint)`         | uint → строка                | `str s = uintToStr(42u)`                    |
| `moreToStr(more)`         | more → строка                | `str s = moreToStr(123456789LL)`            |
| `umoreToStr(umore)`       | umore → строка               | `str s = umoreToStr(987654321ULL)`          |
| `fltToStr(flt)`           | flt → строка                 | `str s = fltToStr(3.14f)`                   |
| `doubleToStr(double)`     | double → строка              | `str s = doubleToStr(2.71828)`              |
| `boolToStr(bool)`         | bool → "true"/"false"        | `str s = boolToStr(true)`                   |

## Модуль DictServer (JSON)

Подключается через `using DictServer;`. Тип `DictHost` – непрозрачный указатель на словарь.

### Загрузка и сохранение

| Функция                | Описание                                     |
|------------------------|----------------------------------------------|
| `load(str)`            | Загружает JSON из файла, возвращает DictHost |
| `save(DictHost, str)`  | Сохраняет словарь в файл                     |

### Получение значений

| Функция                     | Описание                          |
|-----------------------------|-----------------------------------|
| `getStr(DictHost, str)`     | Возвращает строку по ключу        |
| `getInt(DictHost, str)`     | Возвращает int                    |
| `getBool(DictHost, str)`    | Возвращает bool                   |
| `getDouble(DictHost, str)`  | Возвращает double                 |
| `getObj(DictHost, str)`     | Возвращает вложенный словарь (DictHost) |

### Установка значений

| Функция                         | Описание                         |
|---------------------------------|----------------------------------|
| `setStr(DictHost, str, str)`    | Устанавливает строку             |
| `setInt(DictHost, str, int)`    | Устанавливает целое              |
| `setBool(DictHost, str, bool)`  | Устанавливает булево             |
| `setDouble(DictHost, str, double)` | Устанавливает вещественное    |
| `setObj(DictHost, str, DictHost)`  | Устанавливает вложенный словарь |

### Управление ключами и сериализация

| Функция                | Описание                                    |
|------------------------|---------------------------------------------|
| `del(DictHost, str)`   | Удаляет ключ                                |
| `has(DictHost, str)`   | Проверяет существование ключа               |
| `keys(DictHost)`       | Возвращает массив строк с ключами           |
| `toJson(DictHost)`     | Возвращает JSON-представление словаря       |
| `parse(str)`           | Парсит JSON в DictHost                      |
| `freeDict(DictHost)`   | Освобождает память                          |

## Модули и компилятор elyBoots

### Структура проекта

```json
{
    "name": "ProjectName",
    "libs": {},
    "modules": {
        "DictServer": "./modules/math.e"
    },
    "enter": "main.e",
    "stx": {
        "processType": "console"
    },
    "output": {
        "folder-modules": {
            "modules": "*"
        },
        "enter": {
            "name": "app.exe",
            "type": "exe"
        }
    }
}
```

### Сборка проекта

```bash
python ebt.py build
```

Исполняемый файл появится в папке `output/`.

### Создание модуля

```c
// math.e
public func add(a: int, b: int) -> int {
    return a + b;
}
```

Используйте `using math;` в основном коде. Компилятор сгенерирует заголовочный файл и динамическую библиотеку.
